            • java.util.Map artifactory

              Artifactory configurations

              Use configurations defined at JFrogArtifactory.init.

              Note:
              If this value is not provide, the library will try to init a Artifactory object with default configurations.

              Default:
              1 Minute

            • java.lang.Boolean disableArtifactory

              If the pipeline doesn't need artifactory at all.

              Note:

              Default:
              false

            • java.lang.Boolean disableGithub

              If the pipeline doesn't need github at all.

              Note:
              Set this to true to prevent the library to load default GitHub configurations.
              Default:
              false

            • java.lang.Boolean disablePax

              If the pipeline doesn't need pax at all.

              Note:
              Set this to true to prevent the library to load default Pax configurations.
              Default:
              false

            • groovy.lang.Closure extraInit

              Extra init operations

              This closure will execute in 'Init Generic Pipeline' stage, and the pipeline object will be passed to Clusure as arguments.

            • java.util.Map github

              Github configurations

              Use configurations defined at GitHub.init.

              Note:
              If this value is not provide, the library will try to init a GitHub object with default configurations.

            • java.lang.String githubTagPrefix

              Github tag prefix

              If you set a prefix example, then all version tags like v1.2.3 will have a github tag example-v1.2.3.

            • org.zowe.jenkins_shared_library.pipelines.base.models.StageTimeout initForGeneric

              Amount of time allowed for the init pipeline

              Default:
              1 Minute

            • groovy.lang.Closure operation

              The stage operation.

              This closure is used by the various stage methods to perform an operation for the stage in conjunction with some common pre and post operation steps. Additional documentation for this argument will be provided in each command.

              Note:
              Since trait and "@Mixion(GenericStageArgumentsTrait)" doesn't work with CPS, we import this property directly.

            • java.util.Map pax

              PAX server configurations

              Use configurations defined at Pax.init.

              Note:
              If this value is not provide, the library will try to init a Pax object with default configurations.
