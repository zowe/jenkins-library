[Groovy] Class Constants

    • jenkins_shared_library.Constants

    • Constants used by Jenkins Shared Library.

      Note:
      The default account information for various services we are using are also defined here. We strongly recommended the library consumer to use these constants. This will make us really easy to migrate service to another location.
