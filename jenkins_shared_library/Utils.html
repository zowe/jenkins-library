        • static java.util.Map parseSemanticVersion(java.lang.String version)

          Parse semantic version into trunks.

          The result should be a Map with these keys:

          • - major
          • - minor
          • - patch
          • - prerelease
          • - metadata

          Example:
          With input: 1.2.3-alpha.1+20190101010101, the method should return an output of:
          • - major: 1
          • - minor: 2
          • - patch: 3
          • - prerelease: alpha.1
          • - metadata: 20190101010101

          Parameters:
          version - the version string to parse
          Returns:
          a Map with major, minor, path, prerelease and metadata keys.

        • static java.util.Map waitForInput(java.lang.Object jenkinsSteps, java.util.Map args = [:])

          Pause the Jenkins pipeline and wait for end-user input.

          The result should be a Map include these keys:

          • timeout: a Boolean value if the input has reached timeout.
          • proceed: a Boolean value if the pipeline should proceed. It will be true only if the input is Approved to proceed by a user.
          • user: a nullable string represents who approved/rejected the input.

          Example:
          Pause the pipeline for 30 minutes: 
           Map action = Utils.waitForInput(
     this.steps,
     [
         timeout: [time: 30, unit: 'MINUTES'],
         message: "Please verify before continue:"
     ]
 )

          Here is senarios what could be returned:

          • [timeout: true, proceed: false, user: 'TIMEOUT']: No one intervened, so the input reached timeout.
          • [timeout: false, proceed: true, user: 'Jack T. Jia']: Jack clicked on Proceed button.
          • [timeout: false, proceed: false, user: 'Jack T. Jia']: Jack chosed to abort the pipeline.

          Parameters:
          jenkinsSteps - jenkins steps object
          args - arguments for the input:
          • - timeout map of timeout. For example [time: 2, unit: 'MINUTES']
          • - message message of the input
          • - proceedButton proceed button text
          Returns:
          A map represents how the input is handled.
